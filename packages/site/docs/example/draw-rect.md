# Rectangle drawing tool

Drawing tools translate pointer streams into shape edits—see [Lesson 25 — Drawing mode and brush](/guide/lesson-025). Rectangle mode is the simplest test bed for modifiers (Shift/Alt).

Snap to grid or objects from [Lesson 27](/guide/lesson-027) when precision matters.
