该插件提供了一个工具方法，将 [D2] 语法转换成画布的场景图：

```ts
import { parseD2ToSerializedNodes } from '@infinite-canvas-tutorial/d2';

const nodes = await parseD2ToSerializedNodes('x -> y: hello world');
api.runAtNextTick(() => {
    api.updateNodes(nodes);
});
```
