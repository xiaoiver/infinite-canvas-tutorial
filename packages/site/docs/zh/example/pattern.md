# 重复图案（Pattern）填充

**图案** 填充常用于示意图纹理。需控制平铺尺寸、偏移与变换，避免导出分辨率下的摩尔纹，见 [第 17 课](/zh/guide/lesson-017)。

印刷导出宜使用足够分辨率的贴图。
